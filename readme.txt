Substitutes elements from the Super Trooper's Cat Game into a WordPress comments section.

== Description ==

WordPress plugin to interject elements from the [Super Troopers](http://www.imdb.com/title/tt0247745/) "[Cat Game](https://www.youtube.com/watch?v=mXPeLctgvQI)" into post comments. Basically "Now"s become "Meow"s. Do you get it meow?
